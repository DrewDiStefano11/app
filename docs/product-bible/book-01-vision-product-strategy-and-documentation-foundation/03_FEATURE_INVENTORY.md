# Feature Inventory

This inventory captures all known desired, deferred, and future FitCore features from Book 1. It is intentionally broad. Deeper specifications belong in later Books or focused docs under `docs/product`.

Status and priority definitions are in [05_FEATURE_STATUS_AND_PRIORITY_SYSTEM.md](./05_FEATURE_STATUS_AND_PRIORITY_SYSTEM.md).

Legend:

- Priority: P0 foundation, P1 core, P2 expansion, P3 advanced, P4 future/deferred.
- Status: Planned, Deferred, Future, Needs Decision, Implemented.
- Product domain is indicated by each section heading.
